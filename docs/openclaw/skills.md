# Skills

> OpenClaw Skills 是功能扩展包，类似 iOS 的 App，给 AI 助手增加新能力。

## Skills 市场

- 官方市场：https://clawhub.com

## 内置 Skills

| Skill | 说明 |
|-------|------|
| weather | 查询天气预报 |
| healthcheck | 安全审计与系统检查 |
| skill-creator | 创建自定义 Skills |

## 安装 Skill

```bash
openclaw skills install <skill-name>
```
